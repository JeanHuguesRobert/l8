Hello! This is l8. Please look for the results of the test suite in the console.
Please note that, at this point, the test suite includes functional tests only, not unit tests, and is not integrated yet with a standard tool. Expect to see "SUCCESS!!! All tests ok" if the 12 tests work.
If the script l8_browser_test_suite.min.js fails to load, check if it exists.
If not run browserify test/suite.js -o l8_browser_test_suite.js in node_modules/l8 Then copy the result into your usual javascript directory.
To build the minimized version, run uglifyjs l8_browser_test_suite.js -o l8_browser_test_suite.min.js --source-map l8_browser_test_suite.min.js.map -c
If you need a compressed version, run gzip l8_browser_test_suite.min.js
